Guia para a instalação do Electron

Pré-requisitos: É preciso ter o NodeJS instalado, é recomendado a versão LTS
Para ver se o NodeJS foi instalado corretamente, utilize os comandos: node -v e npm -v no seu terminal

Estrutura
